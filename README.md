# TuxShow: Show Control & Mapping App

TuxShow is an open-source, cue-based projection mapping and show control application built specifically for Ubuntu Linux. Designed with educational theater in mind, TuxShow empowers junior high and high school students to produce professional-grade visuals for live stage productions using readily available, affordable hardware.

### 🎭 Why TuxShow?

Professional projection software can be prohibitively expensive for school districts. TuxShow bridges that gap by providing:
* **Accessibility:** Runs on standard Linux-based hardware (including many older laptops and single-board computers).
* **Theatrical Logic:** Designed around a standard cue-list workflow familiar to Stage Managers and Designers.
* **Native Performance:** Built for Ubuntu to ensure stability during live performances.

### ✨ Key Features

* **Cue Engine:** Support for linear playback, timed auto-advancing, and non-linear "GoTo" or "Loop" commands.
* **Projection Mapping:** Integrated Mesh Warping to align projections with physical set pieces or uneven surfaces.
* **Live Control:** Global Pause and Fade-In/Out logic for real-time adjustments during a show.
* **Media Support:** Native handling of .mp4, .webm, .png (with transparency), and more.
* **Batch Editing:** Quickly update properties across multiple cues at once.
* **Workspace Management:** Save entire show designs into a single .TSW file.
* **Multi-Monitor Support:** Dedicated, borderless "Stage" window for external projectors.

### 💻 System Requirements

TuxShow is optimized for the Ubuntu desktop environment.
* **OS:** Ubuntu 22.04 LTS or later (Supports Ubuntu Cinnamon and other flavors).
* **Hardware Minimum:** 4GB RAM, Dual-core processor.
* **Hardware Recommended:** 8GB RAM, Dedicated GPU (for complex warping).
* **Experimental:** Support for Raspberry Pi 4/5 and Orange Pi models.

### 🚀 Getting Started

1. **Download:** Grab the latest `.deb` installer from the Releases page.
2. **Install:** Double-click the file or run `sudo dpkg -i tuxshow.deb` in your terminal.
3. **Launch:** Open TuxShow from your applications menu.
4. **Learn:** Check out our Beginner's Guide to Digital Scenery for tips on your first show.
